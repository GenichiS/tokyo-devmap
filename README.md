# 🏙️ 東京23区 不動産オーナーマップ

東京23区の主要不動産デベロッパー・ビルオーナー **32社・1,145棟** のビルを、インタラクティブな地図上に色分け表示するWebアプリです。

![Developer Map](https://img.shields.io/badge/buildings-1%2C145-blue)
![Developers](https://img.shields.io/badge/developers-32-green)
![Year Coverage](https://img.shields.io/badge/竣工年-86%25-orange)

## ✨ 特徴

- **デベロッパー別表示**: 32社を色分け。サイドバーでON/OFF切替
- **竣工年別表示**: 青(古)→緑→黄→橙→赤(新)のグラデーション
- **タイムラプスアニメーション**: 1925年〜2027年、ビルが建っていく様子を再生
- **ビル詳細ポップアップ**: ビル名・デベロッパー・階数・高さ・竣工年・延床面積を表示
- **OSM建物フットプリント**: 多くのビルは実際の建物形状で描画

## 🛠️ 技術構成

- **地図**: [Leaflet.js](https://leafletjs.com/) + [CARTO Positron](https://carto.com/basemaps/) タイル（グレースケール化）
- **建物形状**: [OpenStreetMap](https://www.openstreetmap.org/) のビルフットプリント
- **データ形式**: GeoJSON（HTML内にインライン埋め込み）
- **依存関係**: CDNからのLeaflet.jsのみ（サーバー不要）
